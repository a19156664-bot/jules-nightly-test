# T2-01: --restore-snapshot CLI追加

## 目的
commander/state_manager.py に --restore-snapshot YYYY-MM-DD オプションを追加し、指定日付のスナップショットから state.yml を復元できるようにする。

## 完了条件
1. python -m commander.state_manager --restore-snapshot 2026-07-20 で commander/state-snapshot-2026-07-20.yml の内容を state.yml に上書き復元する
2. 復元時に stop_reason と error_count は復元元の値をそのまま使用する（自己緩和の禁止に抵触しないよう、値の改変は行わない）
3. 指定日付のスナップショットが存在しない場合はエラーメッセージを出力して終了（exit code 1）
4. 復元成功時は stdout に "Restored from snapshot: YYYY-MM-DD" を出力
5. テストを追加: スナップショットからの復元、存在しない日付の指定、復元後の state.yml 内容検証
